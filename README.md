# 一向聴 基礎講義フラッシュカード

基礎講義の全50問を、ログインなしで反復できるフラッシュカードアプリです。問題は静的データとして収録し、解き直しカードと直近の結果だけをブラウザの `localStorage` に保存します。

## 主な機能

- 全50問・解き直しカードの2モード
- 答えを表示してから「わかった」「解き直しに追加」で自己判定
- わかった率と定着ランク（D〜S）の結果表示
- 全50問の開閉式一覧
- `m`・`p`・`s` 表記の牌姿を承認済み牌画像で表示
- Space / ← / → のキーボード操作

## 開発

Node.js `22.13.0` 以降を使用します。

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm test
```

表示バージョンは `app/lib/flashcards.mjs` の `APP_VERSION` と同期しています。問題・UI・アセットを更新したときは、この値も増やしてください。
