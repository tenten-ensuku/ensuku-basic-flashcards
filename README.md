# 一向聴 基礎講義フラッシュカード

麻雀の基礎講義を、4択クイズとフラッシュカードの両方で反復できる復習アプリです。問題は静的データとして収録し、フラッシュカードの解き直しカードと直近の結果だけをブラウザの `localStorage` に保存します。

## 主な機能

- 7/16「基本序列マスタークイズ」全30問（4択・解説・間違い問題の再挑戦）
- 7/18の30問、7/14・7/2の各50問と解き直しカード
- 管理画面からフラッシュカードと4択クイズ（問題・選択肢・正解・解説）を公開編集
- 答えを表示してから「わかった」「解き直しに追加」で自己判定
- わかった率と定着ランク（D〜S）の結果表示
- 各授業の全問題を確認できる開閉式一覧
- `m`・`p`・`s` 表記の牌姿を承認済み牌画像で表示
- クイズは1〜4・A〜D・Enter、フラッシュカードはSpace・←・→でキーボード操作

## 開発

Node.js `22.13.0` 以降を使用します。

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm test
```

表示バージョンは `app/lib/flashcards.mjs` の `APP_VERSION` と同期しています。問題・UI・アセットを更新したときは、この値も増やしてください。
